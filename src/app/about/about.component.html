VNT CHAT is an application developed for learning purposes. It composes the front-end of a chat application which connects to a back-end server and allows users to exchange messages.

This application is part of Venturus Bootcamp Angular, which happened in 2017 at Venturus Innovation and Technology Center.

The app was built on top of Angular 5 and allowed the developers to exercise different concepts of the framework which are needed to build functional applications.
